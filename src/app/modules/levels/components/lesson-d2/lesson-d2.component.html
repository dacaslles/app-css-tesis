DISPLAY INSIDE - FLEX

Como se mencionó en la lección anterior, un contenedor puede indicar la forma en la cual sus elementos serán desplegados. Los valores "grid" y "flex" son los más utilizados. Flex tiene la cualidad de ofrecer una variedad de opciones presentando gran flexibilidad al momento de indicar el despliegue.
Para utilizar flex basta con indicar este valor en la propiedad display, "display:flex". Inicialmente los elementos serán desplegados de forma lineal iniciando desde la parte izquierda y ocuparán solamente el espacion de su contenido. Para modificar estas posiciones es necesario agregar otros atributos.

flex-direction

Este indica la dirección en la cual se ordenarán los elementos, estos pueden ser cuatro valores:
  • row: ordenamiento predeterminado en el que los elementos se posicionan de izquierda a derecha
  • column: en este ordenamiento los elementos se posicionarán en forma de bloque, equivalente a "display: block" empezando de la parte superior hacia abajo
  • reverse column y row: este valor invierte el orden de los elementos según la forma que se haya indicado, en el caso de reverse-row los elementos serán colocados empezando desde la parte derecha hacia la izquierda y para el valor reverse-column los elementos se colocarán empezando en la parte inferior hacia arriba

<h4>Etiquetas <i>span</i> con propiedad display: block</h4>
<span class="span-verde display-block"> span rojo </span>

}

RETO

Utiliza la propiedad flex y el atributo "flex-direction" para cumplir con lo siguiente:
  • El Distrito Pesadilla debe tener sus casas ordenadas en forma lineal iniciando desde la izquierda

  • El Distrito Vampiro debe tener sus casas ordenadas de forma vertical iniciando desde la parte inferior
