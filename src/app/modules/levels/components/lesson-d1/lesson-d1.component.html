DISPLAY OUTSIDE

La propiedad display en CSS especifica la forma en la que serán desplegados los elementos dentro de un contenedor. De esta forma es posible indicar si los elementos se mostraran en fila, columna, cuadrícula, si estarán centrados o alineados al final, etc. Es posible utilizar esta propiedad para indicar la forma de despliegue del propio elemento o indicar la forma en la que serán desplegados los elementos en su interior, si este se trata como un contenedor. A los valores de display que indican el despliegue del propio elemento se les conoce como "display outside", mientras que los valores que indican el despliegue de los elementos internos se les conoce como "display inside".
Los valores de display outside más utilizados son "block", "inline" y "inline-block".

display: block

Indica que el elemento será desplegado como un elemento "bloque", esto quiere decir que el elemento iniciará suposición en una nueva línea como lo hacen de forma predeterminada los elementos div y los elementos p.

Ejemplo:

Etiquetas span sin modificar su despliegue

span rojo span azul span verde

Etiquetas span con propiedad display: block

span rojo span azul span verde 
                                    
<h4>Etiquetas <i>span</i> sin modificar su despliegue</h4>
<span class="span-rojo"> span rojo </span>
<span class="span-azul"> span azul </span>
<span class="span-verde"> span verde </span>

<h4>Etiquetas <i>span</i> con propiedad display: block</h4>
<span class="span-verde display-block"> span rojo </span>
<span class="span-azul display-block"> span azul </span>
<span class="span-verde display-block"> span verde </span>
                                    
                                
                                    
.span-rojo {
    background-color: red;
}
.span-azul {
    background-color: blue;
}
.span-verde {
    background-color: green;
}
.display-block {
    display: block;
}
