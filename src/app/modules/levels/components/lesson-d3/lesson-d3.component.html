Display inside - Grid

Otra forma de realizar un despliegue de los elementos internos para un contenedor es a través del valor para display "grid". Esta propiedad permite definir el despliegue de componentes en forma de cuadricula utilizando columnas y filas para definir las posiciones.
Para utilizar flex basta con indicar este valor en la propiedad display, "display:grid". Con esta propiedad si es necesario específicar la distrubución de los elementos, para esto existen varias propiedades de grid con las cuales se indica la cantidad y el tamaño de las columnas.

grid-template

La cantidad de filas y columnas se indica a través de las propiedades "grid-template-columns" y "grid-template-row".
  • grid-template-column: es esta propiedad es posible especificar la cantidad de columnas y el tamaño de las columnas. Existen varias formas de indicarlo, puede ser utilizando la instrucción repeat la cual establece las mismas propiedades para todos los elementos o indicar el tamaño de cada columna en la fila.
  • grid-template-row: esta propiedad indica las características que tendrán las filas del contenido, puede especificar la cantidad y el tamaño de las filas de la misma forma como se especifican las columnas.
