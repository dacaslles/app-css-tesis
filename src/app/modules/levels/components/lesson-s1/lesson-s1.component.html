SELECTORES

En CSS los selectores son utilizados para delimitar a los componentes que serán afectados por un conjunto de atributos. Existen 3 tipos de selectores: por etiqueta, por clase y por identificador.
  • Etiqueta: Se indica la etiqueta (tag) de html que será afectada. Los estilos serán aplicados a todas las etiquetas que se indique.
  • Clase: A la etiqueta se le indica las clases que serán aplicadas para modificar su estilo. No importa el tipo de etiqueta, si tiene una clase indicada esta será afectada por los atributos que tenga.
  • Identificador: En el archivo CSS se indica el nombre de un elemento y solamente ese elemento será afectado por los atributos que allí se especifiquen. Este es el selector más particular que existe ya que solo afectará a un elemento en todo el código.

RETO

Los selectores serán utilizados de la siguiente forma: los nombres son equivalentes al selector "identificador" y el tipo es equivalente al selector "clase".
